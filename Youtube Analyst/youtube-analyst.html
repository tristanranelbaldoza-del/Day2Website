Under the hood
01 · Fetch
youtube_client.py

Pulls last-week videos, views, likes, subscriber counts from YouTube Data API v3.

02 · Write
sheets_writer.py

Appends a dated tab to the Google Sheet with per-video metrics and channel totals.

03 · Build
slides_builder.py

Creates a 3-slide Google Slides deck — cover, metrics, top video of the week.

04 · Send
gmail_sender.py

Emails the summary via Gmail with links to the Sheet and Slides deck.

First run opens a browser for Google OAuth consent. Subsequent runs stay silent — token cached in token.json. See README.md for Google Cloud setup.
